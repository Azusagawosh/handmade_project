# 💎 Maison Crystal - Handmade Маркетплейс

**Maison Crystal** — веб-платформа для продажи изделий ручной работы. Мастера выставляют товары, покупатели ищут, оценивают и покупают уникальные handmade-изделия.

---

## 🛠 Технологии

| Категория | Технологии |
|-----------|-------------|
| **Backend** | Python 3.14, Django 6.0, DRF 3.17, djoser, django-filter, SQLite |
| **Frontend** | React 18, React Router, Bootstrap 5, Axios, React Icons |

---

## ✅ Функционал

### 👤 Гость
- Просмотр каталога, поиск, фильтр по категориям
- Карточка товара, отзывы, рейтинг
- Страница "Обзор" с характеристиками

### 👤 Пользователь
- Регистрация / вход
- Редактирование профиля (email, имя, телефон)
- Загрузка аватарки
- Избранное, отзывы и оценки

### 🛍️ Продавец
- Добавление / редактирование / удаление товаров (с фото)
- Просмотр своих товаров
- Просмотр отзывов на свои товары

### 👑 Администратор
- Управление пользователями (смена ролей)
- Удаление любых товаров и отзывов
- Django Admin + React Админ панель
